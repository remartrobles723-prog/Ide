## Overview
This major release introduces new features, important updates, and several key fixes aimed at improving stability and the overall development workflow.

---

## Full Changelog

### Major Changes
1. Added support for newer AGPs.  
2. Updated Gradle Tooling API to version 8.9.  
3. Added Kotlin Language Server integration.  
   - To activate: go to `Preferences → Editor → Language Server Protocol → Kotlin → Kotlin Language Server`, then click **Download**.  
4. Added partial support for Google M3 layouts and views.  
5. Reworked the template system, now managed by the ACS Template System.  
   - Thanks to [Danielk10](https://github.com/Danielk10/Up-to-Date-Android-Studio-and-android-code-studio-Templates.git) for providing the templates.  
6. Submodule Maker now creates library modules instead of raw modules.  
7. Added AI Agent chat mode, long-press the send button to switch between chat and file modification modes.

---

### Major Fixes
1. Fixed an OutOfMemory issue when opening large projects.  
   - [Issue #8](https://github.com/AndroidCSOfficial/android-code-studio/issues/8#issue-3461037344)  
2. Fixed XML layout preview detection.  
3. Fixed App Log plugin integration (now properly attaches to user projects).  
4. Fixed sidebar overlapping issue.
